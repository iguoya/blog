# 工科与数学系线性代数的差别




| 数学系                | 工科需要                 |
| --------------------- | ------------------------ |
| 小矩阵                | 大矩阵                   |
| 小矩阵的经典理论      | 只讲对大矩阵有用的理论   |
| 手工推演              | 计算机和软件包           |
| 以符号推理为主        | 要求数字结果并有实际意义 |
| 强调N维空间和抽象思维 | 强调3维空间和形象概念    |
| 少量的小概念题        | 大量的从简到繁的应用实例 |
